The program makes use of Python 3. 

To run letterCount.py, navigate via the command line to the directory holding the script. Using the command: 
	python3 letterCount.py [filename.txt]

Where filename.txt is the path/name of the desired text file (located within the same directory). If the file name parameter is absent, a prompt will be displayed asking for the user to supply a file name.

Example use of command:
	python3 letterCount.py example.txt
